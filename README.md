# GraphRAG AI Agent Common Framework

GraphRAG 기반 AI Agent 개발 시 반복적으로 필요한 공통 기능을 프레임워크화하기 위한 프로젝트입니다.

## 목표

- GraphRAG, RAG, LangGraph Agent Workflow 공통 구조 개발
- Vector Store 및 Graph Store 추상화
- 도메인별 Entity/Relation Schema 확장 구조 제공
- 기존 프로젝트(`VectorMoon`, `Sol-Bat`, `accountBook`, `lotto`)의 공통 기능을 `vm-common-core` 중심으로 정리
- 신규 AI 서비스 개발 시 재사용 가능한 기반 프레임워크 제공
